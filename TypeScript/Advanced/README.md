### Why Learn Advanced TypeScript?

Advanced TypeScript concepts are essential for building scalable and maintainable applications. Angular relies heavily on features like decorators, modules, and advanced types.

---

### Usage

To run TypeScript files:
1. Install TypeScript globally:
   ```bash
   npm install -g typescript
   ```
2. Compile TypeScript files to JavaScript:
    ```
    tsc <filename>.ts
    ```
3. Run the compiled JavaScript file:
    ```
    node <filename>.js
    ```
