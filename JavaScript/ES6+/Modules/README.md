# Modules in JavaScript

This folder demonstrates how to use JavaScript ES6 modules for better code organization and reusability.

### Topics Covered
1. Exporting Functions, Variables, and Classes
2. Importing Modules
3. Default Exports
4. Named Exports
5. Combining Exports

---

### File Structure
- **math.js**: Contains math-related functions.
- **greetings.js**: Contains greeting-related functions.
- **app.js**: Main file where modules are imported and used.

---

### Usage
Run the `app.js` file in a browser or Node.js environment that supports ES6 modules.
